### Theory

A function in Python is a block of organized and reusable code that performs a specific task. Functions help in dividing a large program into smaller modules, which improves readability and reduces code repetition.

In Python, functions are defined using the `def` keyword followed by the function name and parentheses. A function may take parameters as input and may return a value using the `return` statement.

Functions can be classified as:

• Built-in Functions – Predefined functions provided by Python such as `print()`, `input()`, `len()`, etc.

• User-defined Functions – Functions created by the programmer using the `def` keyword.

Functions improve modularity, reusability, and maintainability of programs.

---

### Functions Used

• `def` – Used to define a function.

• `return` – Used to return a value from the function.

• `input()` – Used to accept user input.

• `print()` – Used to display output.
